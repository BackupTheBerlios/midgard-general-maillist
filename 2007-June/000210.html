Also externe Pakete werden (bei einer halbwegs aktuellen Distribution) 
eigentlich nicht gebraucht. 

Das rpm-Paket für SuSE 10.1, i586 habe ich auf einer 
Standardinstallation erstellt. 
Es werden noch die development-Pakete für gtk-benötigt, die 
sollten aber bei einer Standarddistribution dabei sein.

Für das RPM bin ich wie folgt vorgegangen:

cd ManuProC_Base
aclocal -I macros
./autogen.sh [--prefix=$ZIEL; wenn nicht /usr/local]
make install

cd ../GtkmmAddons
aclocal
./autogen.sh

cd ../ManuProC_Widgets
aclocal -I macros
./autogen.sh

cd ../midgard
aclocal
./autogen.sh [--prefix=$ZIEL ; wenn nicht /usr/local]
make install


bis dann,
